객체 - 자료형
	  - 사물, 대상  (Object)
	  - 상호작용 
			- 특성 - 속성, 변수 
			- 행위 - 함수, 메서드 
			
	-  속성, 메서드로 구성되어 있는 자료형
	
	자바스크립트에서 객체 정의방법 
	{
		속성 : 값, 
		속성 : 값, 
		....
		
		속성 : 값
	}
	
	값 - 원시타입(숫자, 문자, /...) + 객체(함수 객체도 포함)
	
	
	객체의 속성에 접근 -  마침표(.) 연산자
	person.age
	person.printAge();
	
	객체의 속성을
		추가
		- 없는 속성에 값을 대입 
	
	    변경  
		- 이미 정의되어 있는 속성에 값을 대입
		
	    삭제
		 delete 연산자 
		 delete 객체변수.속성명;
	
	
	생성자 
		- 객체를 생성하는 역할
		- 객체를 정의하는 틀
		
		function 생성자명(인수... 인수.. ) {
			this.속성명 = 값(원시타입 + 객체);
			..
			..
			this.속성명 = 값... 
		}
		
		- 생성자의 인수 - 초기화 역할
		
		- 객체를 생성하려면 ... new 연산자  필요
		
		this ->생성된 인스턴의 주소를 내부에서 가리키는 연산자
		
		
		메모리
			- 데이터영역 - 작성한 코드
			- 스택영역 - 함수가 호출될때 실행되는 메모리 영역 
						  - 변수
			- 힙영역 - 인스턴스가 생성되는 메모리 영역 
			
		
		- prototype
				.constructor  - 생성자 원본 -> new 연산자 가능
				.[[Prototype]] - 상속관계 
				
				
		- { ... } -> 생성된 객체(new Object())
		
		- 생성자명 -> 앞글자가 대문자인 파스칼 케이스
		- 함수명 -> 앞글자 소문자, 각 단어 시작마다 대문자 - 카멜케이스
