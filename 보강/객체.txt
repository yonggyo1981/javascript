객체 - 자료형
	  - 사물, 대상  (Object)
	  - 상호작용 
			- 특성 - 속성, 변수 
			- 행위 - 함수, 메서드 
			
	-  속성, 메서드로 구성되어 있는 자료형
	
	자바스크립트에서 객체 정의방법 
	{
		속성 : 값, 
		속성 : 값, 
		....
		
		속성 : 값
	}
	
	값 - 원시타입(숫자, 문자, /...) + 객체(함수 객체도 포함)
	
	
	객체의 속성에 접근 -  마침표(.) 연산자
	person.age
	person.printAge();
	
	객체의 속성을
		추가
		- 없는 속성에 값을 대입 
	
	    변경  
		- 이미 정의되어 있는 속성에 값을 대입
		
	    삭제
		 delete 연산자 
		 delete 객체변수.속성명;
	
	
	생성자 
		- 객체를 생성하는 역할
		- 객체를 정의하는 틀
